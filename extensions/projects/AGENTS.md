# projects

Project initialization extension for Pi.

## Layout

- `commands/init.ts` - /projects:init command registration
- `commands/init/` - Wizard and supporting modules (catalog scanner, installer, nix/AGENTS.md prompt builders, scanner)
- `commands/settings.ts` - /projects:settings command
- `config.ts` - Extension config (catalog paths, scan depths)
- `index.ts` - Entry point
