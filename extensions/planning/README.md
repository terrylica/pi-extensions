# Planning

Commands for turning conversations into implementation plans and executing saved plans.

## Features

- **Command**: `/save-as-plan` - creates a structured plan from the current conversation
- **Command**: `/execute-plan` - selects and runs a saved plan from `.agents/plans/`

## Usage

### Command (interactive)

Run `/save-as-plan` to generate a plan and `/execute-plan` to run one.
